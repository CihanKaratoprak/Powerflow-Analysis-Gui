# Power Flow Analysis GUI (AC/DC) + Short Circuit

This repository contains the release information of a power systems analysis desktop application developed as an R&D-focused engineering project.

## Features
- AC power flow
- DC load flow (optional method)
- Short-circuit analysis
- Scenario save/load
- Automatic line parameter calculation (R–X) based on conductor selection and line length

## Download & Install
Go to **Releases** and download the latest `PowerFlowAnalysis_Setup_*.exe`.
